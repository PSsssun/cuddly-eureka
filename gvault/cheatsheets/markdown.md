# markdown cheatsheet  

## 1. Tables  

### 1.1 Format1

```
|Syntax|Description|
|:-- | :--|
|Header | Title|
|Paragraph | Text|
```
shows: 
|Syntax|Description|
|:-- | :--|
|Header | Title|
|Paragraph | Text|

### 1.2 Format2

```
|Syntax|Description|
|----|----|
|Header|Title|
|Paragraph|Text|
```
shows:
|Syntax|Description|
|----|----|
|Header|Title|
|Paragraph|Text|

## 2. Code Blocks  

### 2.1 Fenced Code Blocks

```
{
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
}
```

### 2.2 Syntax Highlighting

```json
{
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
}
```
or
```javascript
{
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
}
```
